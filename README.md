Developed a full-stack real estate web application using the MERN stack that allows users to browse, search, and list properties for sale. The platform helps bridge the gap between property buyers and real estate agents by providing detailed property listings and an intuitive user experience.

Key Features

Displayed property listings with detailed information for properties available for sale

Implemented property search functionality to help users quickly find relevant listings

Enabled users to add and list properties for sale through a user-friendly interface

Implemented property rating functionality to improve engagement and trust

Tech Stack

Frontend: React.js

Backend: Node.js, Express.js

Database: MongoDB
